# 🏠 Home Microfrontend - TEAM 4

This is the Home page microfrontend for the MFE Workshop ecommerce application.

## 🎯 Features

- Hero banner with promotional content
- Category browsing section
- Hot deals with featured products
- Promotional banners (Gift Cards, Free Shipping)
- Newsletter subscription section

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server on port 3004
npm start
```

## 📦 What This MFE Exposes

- `./Home` - Main home page component with all sections

## 🎨 Customization Ideas

1. Add a carousel for hero banners
2. Connect categories to actual product filtering
3. Add countdown timers for limited offers
4. Implement real newsletter integration
5. Add testimonials section
6. Add seasonal themes

## ✅ Verification

1. Open http://localhost:3004 to see your component standalone
2. Changes will reflect automatically in the Host (http://localhost:3000)

## 🔧 Technical Details

- **Port**: 3004
- **Exposes**: `./Home`
- **Framework**: React 18
- **Styling**: Inline styles + Tailwind CSS
